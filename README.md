# Scoop Bucket for witnessd

This is the official Scoop bucket for [witnessd](https://github.com/writerslogic/witnessd) - Cryptographic Authorship Witnessing.

## Installation

```powershell
scoop bucket add writerslogic https://github.com/writerslogic/scoop-bucket
scoop install witnessd
```

## Updating

```powershell
scoop update witnessd
```
